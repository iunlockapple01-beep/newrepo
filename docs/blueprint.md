# **App Name**: HTML Insights

## Core Features:

- HTML Upload: Allow users to upload their HTML code through a simple UI component.
- Code Analysis: Use AI to analyze the HTML code for common errors and potential improvements (e.g., accessibility issues, broken links, missing alt text, opportunities for semantic HTML). Use it as a tool and incorporate reasoning to decide whether the issue is relevant for the website
- Suggestion Display: Clearly present suggestions and identified issues to the user, grouped by category and severity.
- Improvement Suggestions: Offer AI-powered code improvement suggestions directly within the app for users to apply.
- Download Enhanced Code: Lets users download the analyzed and enhanced HTML for use.

## Style Guidelines:

- Primary color: A deep sky blue (#42A5F5), evoking clarity and trustworthiness.
- Background color: Light grayish-blue (#ECEFF1), providing a clean and unobtrusive backdrop.
- Accent color: A vibrant violet (#AB47BC), to draw attention to key interactive elements.
- Body and headline font: 'Inter', a sans-serif font, should be used to enhance the clean aesthetic. It ensures optimum readability, especially when displaying code.
- Code Font: Use 'Source Code Pro' for all displayed code snippets
- Use simple, modern icons to represent different types of suggestions (e.g., accessibility, performance, SEO).
- A clean and well-organized layout is essential for presenting complex information clearly. Prioritize readability and intuitive navigation.
- Subtle transitions and animations can enhance the user experience, but avoid excessive or distracting effects.
